# C++のバージョン
コードはC++17を使用します。
廃止予定の機能や、最新の言語バージョンで廃止している機能の使用は避けてください。
# ファイル
コーディングには以下のファイルを使用します。
インライン関数はヘッダーファイルの最下部に記述してください。
* .cpp
* .h
# インクルードガード
```#pragma once```を使用してください。
# 名前空間
ドキュメントを名前空間をもとに生成しているため、コードは基本的に名前空間の中に置くようにしてください。
```c++
namespace ob::platform{
    class Hoge;
}// namespace ob::platform
```
